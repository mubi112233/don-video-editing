## 📞 SUPPORT & TROUBLESHOOTING

If pages aren't indexing:
1. Check `SEO_TROUBLESHOOTING.md`
2. Check GSC Coverage report
3. Verify robots.txt allows page
4. Check page has `index, follow` in robots meta
5. Verify canonical URL correct
6. Wait 1-2 weeks, then resubmit

If rankings aren't improving:
1. Check keyword difficulty
2. Analyze top ranking pages
3. Create more content
4. Build more backlinks
5. Optimize on-page elements

If traffic drops:
1. Check for penalties in GSC
2. Check for broken links
3. Verify no noindex tags added
4. Check server status
5. Review recent changes
